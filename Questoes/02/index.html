Desafio Prático - Algoritmos

DESCRIÇÃO NARRATIVA

  1. Fazer um café coado

Para preparar um café coado, primeiro pegue o pacote de café e abra-o. Em seguida, utilize uma jarra ou recipiente para colocar a quantidade necessária de água. Coloque a água para ferver no fogão e aguarde até atingir o ponto de fervura.
Enquanto isso, prepare o coador adicionando o pó de café na quantidade desejada. Após a água ferver, posicione uma garrafa térmica ou outro recipiente abaixo do coador. Em seguida, despeje lentamente a água quente sobre o pó de café no coador.
A água passará pelo pó, realizando a extração do café, e a bebida já coada será armazenada na garrafa térmica, pronta para servir.

  1. Configurar o despertador no celular

Antes de dormir, pegue o celular, abra o aplicativo de relógio e acesse a opção de configuração do despertador. Em seguida, defina o horário desejado para acordar no dia seguinte e ajuste outras configurações, se necessário. Após concluir, salve ou ative o alarme para garantir que o despertador tocará no horário programado.

  1. Ligar um computador

Para ligar o computador, primeiro certifique-se de que ele está conectado corretamente à tomada ou à fonte de energia. Após confirmar que há alimentação elétrica, pressione o botão Power para iniciar o equipamento. Aguarde alguns instantes até que o sistema seja carregado e o computador esteja pronto para uso.

  1. Carregar o celular

Para carregar o celular, conecte o carregador a uma tomada elétrica e, em seguida, conecte o cabo ao aparelho. Após a conexão, verifique se o carregamento foi iniciado corretamente e aguarde até que a bateria atinja o nível desejado ou seja totalmente carregada.

  1. Assistir a um filme em casa

Para assistir a um filme em casa, primeiro é necessário ligar a TV, notebook ou outro dispositivo compatível e certificar-se de que ele está conectado à internet. Em seguida, acesse uma plataforma de streaming utilizando sua conta, navegue pelo catálogo disponível, escolha o filme desejado e inicie a reprodução. Depois disso, basta aproveitar o momento e assistir ao filme.

  1. Regar uma planta

Para regar uma planta, é necessário pegar um regador e enchê-lo com água. Em seguida, leve-o até a planta e despeje a água cuidadosamente na terra ao redor dela, garantindo a hidratação adequada sem excesso.

  1. Fazer uma pipoca

Para fazer pipoca de micro-ondas, primeiro é necessário adquirir um pacote de pipoca em um supermercado. Em seguida, ligue o micro-ondas, retire a embalagem externa do produto (quando aplicável) e coloque o pacote de pipoca dentro do aparelho conforme as instruções da embalagem. Feche a porta do micro-ondas, selecione a função adequada ou ajuste o tempo recomendado e inicie o preparo. Após o tempo necessário, a pipoca estará pronta para consumo.

  1. Participar de uma reunião online

Para participar de uma reunião online, primeiro é necessário confirmar a data e o horário em que ela acontecerá. Em seguida, escolha um local adequado, preferencialmente silencioso, bem iluminado e com poucas interrupções, como uma sala reservada.
No dia da reunião, verifique se a câmera, o microfone e o áudio do computador ou dispositivo estão funcionando corretamente. Também é importante confirmar que a conexão com a internet está estável para evitar falhas durante a participação.
Após realizar essas verificações, localize o link de acesso enviado para a reunião e clique nele no horário agendado para entrar na sala virtual. Depois disso, basta participar da reunião normalmente, mantendo atenção às orientações e interações realizadas.

  1. Criar uma lista com as tarefas da semana

Para criar uma lista de tarefas da semana, primeiro é importante planejar quais atividades, compromissos ou objetivos deverão ser realizados ao longo dos dias. Após definir a programação semanal, pegue um papel ou outro meio de anotação e organize os dias da semana separadamente.
Em seguida, divida cada dia em tópicos ou horários, registrando as tarefas que precisam ser executadas em cada período. Escreva as informações de forma clara para facilitar o acompanhamento das atividades ao longo da semana.
Essa organização ajuda a visualizar melhor os compromissos, priorizar tarefas e manter uma rotina mais estruturada.

  1. Assistir a um vídeo do Diego no canal da Rocketseat 💜

Para assistir a uma videoaula, primeiro é necessário certificar-se de que o computador está ligado e conectado à internet. Em seguida, abra o navegador e acesse a plataforma da Rocketseat.
Após acessar o site, realize o login informando seu usuário e senha. Em seguida, localize a formação Lógica de Programação, acesse a seção de aulas e selecione a aula desejada para iniciar a reprodução.
Depois disso, basta acompanhar o conteúdo e realizar as atividades propostas durante o aprendizado.

{

  1. Verificar se o número é par ou ímpar
programa 
{
  funcao inicio() 
  {
    inteiro numero = 0, resto = 2

    escreva("Digite um número inteiro qualquer: ")
    leia(numero)

    se(numero % resto == 0)
    {
      escreva("O número é par!")
    }
    senao
    {
      escreva("O número é impar!")
    }
  }
}
  1. Verificar se número é positivo, negativo ou zero
programa 
{
  funcao inicio() 
  {
    inteiro numero = 0

    escreva("Digite um número inteiro qualquer: ")
    leia(numero)

    se(numero > 0)
    {
      escreva("O número é positivo!")
    }
    senao se(numero < 0)
    {
      escreva("O número é negativo!")
    }
    senao
    {
      escreva("O número é zero!")
    }
  }
}
  1. Calcular média de 3 notas
programa 
{
  funcao inicio() 
  {
    inteiro divisor = 3
    real nota01 = 0, nota02 = 0, nota03 = 0, media = 0

    escreva("Digite a primeira nota: ")
    leia(nota01)

    escreva("Digite a segunda nota: ")
    leia(nota02)

    escreva("Digite a terceira nota: ")
    leia(nota03)

    media = (nota01 + nota02 + nota03) / divisor

    escreva("A media das notas é: ", media)
  }
}
  1. Verificar se aluno foi aprovado (com média ≥ 5)
programa 
{
  funcao inicio() 
  {
    inteiro divisor = 3
    real nota01 = 0, nota02 = 0, nota03 = 0, media = 0

    escreva("Digite a primeira nota: ")
    leia(nota01)

    escreva("Digite a segunda nota: ")
    leia(nota02)

    escreva("Digite a terceira nota: ")
    leia(nota03)

    media = (nota01 + nota02 + nota03) / divisor

    escreva("A media das notas é: ", media)

    se(media < 5)
    {
      escreva("\nO aluno foi reprovado!")
    }
    senao
    {
      escreva("\nO aluno foi aprovado!")
    }
  }
}

}
